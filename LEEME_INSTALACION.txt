RIFITA FULL NETLIFY - SIN APPS SCRIPT COMO WEB APP
==================================================

Esta versión usa:
- Netlify como sitio público y backend.
- Google Sheets API + Google Drive API con Service Account.
- Google Sheets como base de datos.

Ya NO usa iframe, NO usa Google Sites y NO depende de que Apps Script deje entrar anónimos.

ARCHIVOS
--------
public/index.html        App principal Rifita
public/admin.html        Administrador de rifas
public/app.js            Lógica de la app principal
public/admin.js          Lógica del administrador
public/style.css         Estilos
netlify/functions/api.js Backend/API
netlify.toml             Configuración Netlify
package.json             Dependencias

PASO 1 - GOOGLE CLOUD
---------------------
1) Crear un proyecto en Google Cloud.
2) Activar estas APIs:
   - Google Sheets API
   - Google Drive API
3) Crear una Service Account.
4) Crear una key JSON para esa Service Account.

PASO 2 - COMPARTIR ARCHIVOS CON LA SERVICE ACCOUNT
--------------------------------------------------
En el JSON vas a ver un email parecido a:

rifita-bot@tu-proyecto.iam.gserviceaccount.com

Compartí con ese email como EDITOR:
- La planilla MASTER donde está CONFIG_RIFAS.
- La planilla TEMPLATE, si vas a usar plantilla.
- La carpeta raíz RIFITA, si querés que cree carpetas dentro de tu Drive.

PASO 3 - VARIABLES EN NETLIFY
-----------------------------
En Netlify, en el sitio:
Site configuration > Environment variables

Agregar:

GOOGLE_SERVICE_ACCOUNT_EMAIL = email de la service account
GOOGLE_PRIVATE_KEY = private_key del JSON
MASTER_SS_ID = ID de la planilla master
TEMPLATE_SS_ID = ID de la plantilla de rifa (opcional, recomendado)
RIFITA_ROOT_FOLDER_ID = ID de la carpeta RIFITA en Drive (opcional, recomendado)
ADMIN_APP_PIN = PIN maestro para entrar a /admin.html

IMPORTANTE PRIVATE KEY:
- Copiala completa desde el JSON, incluyendo:
  -----BEGIN PRIVATE KEY-----
  -----END PRIVATE KEY-----
- Si Netlify la guarda en una sola línea con \n, también funciona.

PASO 4 - SUBIR A GITHUB Y NETLIFY
---------------------------------
Subí la carpeta completa a GitHub:

git add .
git commit -m "Rifita full Netlify"
git push

Después en Netlify:
Add new project > Import from Git > elegir repo.

Configuración:
Build command: vacío
Publish directory: public
Functions directory: netlify/functions

PASO 5 - PROBAR API
-------------------
Abrí:

https://TU-SITIO.netlify.app/.netlify/functions/api

Tiene que devolver JSON con service: "Rifita Netlify API".

ESTRUCTURA DE SHEETS
--------------------
MASTER: hoja CONFIG_RIFAS con columnas:
CODIGO, NOMBRE, SHEET_ID, PIN_ADMIN, ACTIVA, FECHA, FOLDER_ID, FOLDER_URL

Cada rifa usa hojas:
LISTAS
VENTAS
SORTEO_GANADORES
